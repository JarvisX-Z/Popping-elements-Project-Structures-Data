# Bitcoin Wallet C++ Project

## Overview
A simple C++ Bitcoin wallet using Bitcoin's core-style logic to generate keys and addresses.

## Recent Changes
- Initialized project with `cpp-clang20` module.
- Planned implementation of Bitcoin wallet logic.

## Project Architecture
- `main.cpp`: Entry point for wallet operations.
- `Makefile`: Build instructions.
- `replit.nix`: System dependencies.
